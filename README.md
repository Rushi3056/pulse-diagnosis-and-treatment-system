# Pulse Diagnosis and Treatment System

## Overview
The Pulse Diagnosis and Treatment System is an ongoing group research project focused on developing a non-invasive pulse monitoring concept using ESP32, PPG/ECG sensors, and embedded systems. The project combines traditional Nadi Pariksha concepts with modern sensor-based analysis.

## Objective
- To monitor pulse characteristics using biomedical sensors.
- To explore a non-invasive approach for pulse analysis.
- To integrate traditional health concepts with modern embedded technology.
- To design a practical and low-cost prototype for research purposes.

## Technologies Used
- ESP32
- PPG Sensor
- ECG Sensor
- Arduino IDE
- Embedded Systems
- Sensors and Interfacing
- PCB Assembly and Testing

## Features
- Non-invasive pulse monitoring concept
- Sensor-based data acquisition
- Embedded system implementation using ESP32
- Research-oriented health monitoring approach
- Integration of traditional and modern analysis methods
